TWM exporter for blender
------------------------

Exporter for twm (model file format with skeletons) and twa (animation file format)
files. twm and twa files are optimized for use in the DT2 game engine.

Copy the io_scene_twm directory to 

	blender.app/Contents/MacOS/2.57/scripts/addons
	
and then enable it.
